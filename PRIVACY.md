# Privacy Policy

**design-library-plugin** does not collect, store, or transmit any personal data or usage information.

All processing happens locally on your machine:

- Brand design systems are downloaded from public GitHub repositories and stored in your local filesystem.
- Hook scripts read your project files and prompt text locally to detect project type and design intent.
- No analytics, telemetry, or tracking of any kind is included.
- No data is sent to any external server other than GitHub (to clone design data during sync).

For questions, open an issue at https://github.com/zeta92/design-library-plugin/issues
